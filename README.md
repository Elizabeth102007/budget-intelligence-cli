# Budget Intelligence System

The **Budget Intelligence System** is a console-based personal finance management application built with Python. Unlike a traditional expense tracker, this application not only records financial transactions but also analyzes spending patterns, compares expenses across months, evaluates budgets, forecasts potential overspending, and generates detailed financial reports.

The project demonstrates how Object-Oriented Programming (OOP), file persistence, data analysis, and Python's standard library can be combined to build a practical budgeting and financial intelligence system.

## How It Works

* Users record income and expense transactions.
* Transactions are stored in a CSV file for persistent storage.
* Budget limits are managed separately using a JSON configuration file.
* Financial reports are generated by analyzing transaction history.
* Spending is compared against user-defined budget limits.
* Monthly spending trends and savings rates are calculated automatically.
* Forecasts estimate whether current spending habits will exceed budget limits before the end of the month.
* Reports can be exported as formatted text files for future reference.

## Project Structure

The application is organized using object-oriented design:

### Transaction

Represents a single financial transaction.

Responsibilities include:

* Storing transaction information
* Validating transaction amounts
* Converting objects to dictionaries
* Creating objects from CSV records

### Budget

Manages budget limits for expense categories.

Responsibilities include:

* Loading budget data
* Saving budget data
* Updating category limits
* Removing budget limits
* Detecting overspending

### Report

Performs financial analysis and reporting.

Responsibilities include:

* Balance calculation
* Monthly spending analysis
* Savings rate calculation
* Budget comparisons
* Spending forecasts
* Report generation

## Data Storage

### Transactions

Transactions are stored in:

```text
intelligence_transactions.csv
```

Each transaction contains:

* Transaction type
* Category
* Amount
* Description
* Date

### Budget Limits

Budget settings are stored in:

```text
intelligence_budget.json
```

Each category can have its own configurable spending limit.

### Exported Reports

Financial reports are exported as:

```text
intelligence_report.txt
```

The report contains a complete financial summary, including balances, monthly comparisons, savings rates, budget analysis, and spending forecasts.

## How To Run

### Interactive Mode

```bash
python budget_intelligence.py
```

### Generate Report Directly

```bash
python budget_intelligence.py --report
```

This exports a complete financial report without opening the interactive menu.

## Financial Analytics Included

### Balance Tracking

The application automatically calculates:

* Total income
* Total expenses
* Current balance

### Monthly Spending Comparison

The tool compares expenses between the two most recent months and reports whether spending has increased, decreased, or remained unchanged.

### Savings Rate

The application calculates savings using the formula:

```text
(Total Income − Total Expenses) / Total Income × 100
```

### Category vs Budget

For every expense category, the system displays:

* Actual spending
* Budget limit
* Amount over budget

### Overspending Forecast

Using the user's current spending rate, the application predicts whether a budget category is likely to exceed its limit before the end of the month.
